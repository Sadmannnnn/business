README: Лендинг курса "Как начать бизнес с нуля"
Описание проекта
Одностраничный сайт (лендинг) для продвижения бесплатного курса по запуску интернет-магазина с нуля. Лендинг предназначен для привлечения потенциальных клиентов и конвертации их в подписчиков курса через Telegram-бота.

Основные характеристики
Целевая аудитория: Начинающие предприниматели, желающие запустить интернет-магазин

Основное предложение: Бесплатный курс из 4 видеоуроков

Потенциальный доход: От 80 000 до 2 000 000 рублей в первый месяц

Основной CTA: Регистрация через Telegram-бота
